## 에이전트에게 위임한 작업의 범위

- `HealthBar` 공통 컴포넌트 구현 및 플레이어/적 컬러 분리
- `BattleHUD`에 층, 양측 HP, 적 공격 타이머(시계 + 게이지) 구성
- `QuestionPanel`에 문제 타이머(시계 + 게이지), 문제 본문 렌더링 구성
- 긴 문제 대응: 문제 본문 영역 max-height + 내부 스크롤
- 문제 복사 방지: 선택/복사/드래그/우클릭 차단

## 위임 판단 근거

시각 요소와 상태 반영이 강하게 결합된 UI라 구현 속도는 에이전트 이점이 크고,
디자인 정합은 반복 피드백으로 조정하는 방식이 적합했다.

## 에이전트 산출물 검증 방법

1. 체력바 검증
- 플레이어/적 체력바 색상 분리
- HP 비율 변화 시 게이지/텍스트 동기화

2. 타이머 검증
- 적 공격 타이머와 문제 타이머가 각각 별도 게이지로 감소
- 위험 구간 색상 전환 확인

3. 레이아웃 검증
- 전체 화면에서 주요 요소가 한 화면에 유지
- 긴 문제에서 에디터 영역이 과도하게 줄어들지 않는지 확인

4. 정적 검증
- `pnpm build` 통과

## 에이전트가 잘못된 방향을 제시했을 때 교정 방법

- 과거 문서의 "유형 배지/assertNever" 중심 설명은 현행 디자인과 다르므로 제거하고,
  현재 구현(타이머 게이지, 복사 방지, 스크롤 고정) 중심으로 문서를 갱신했다.
