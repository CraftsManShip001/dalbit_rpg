# T04 — 게임 상태 스토어

## 전제 파일 (T03 완료)

- `src/lib/runWithWorker.ts` 구현 완료
- `src/store/gameStore.ts` 빈 스텁 존재, 이 태스크에서 구현

## 목표

Zustand로 게임 전체 상태를 정의하고 액션을 구현한다.
HP 계산, 상태 전환은 이 스토어에서 하지 않고 훅(`useGameEngine`)에서 담당한다.

## 구현

`src/store/gameStore.ts`:

```ts
import { create } from 'zustand'

export type QuestionType = 'quick' | 'copy' | 'algorithm'

export type TurnPhase =
  | 'idle'
  | 'questioning'
  | 'judging'
  | 'result'
  | 'gameover'
  | 'stageclear'

interface GameState {
  playerHp: number
  playerMaxHp: number
  enemyHp: number
  enemyMaxHp: number
  stage: number
  turnPhase: TurnPhase
  currentQuestionType: QuestionType | null

  damagePlayer: (amount: number) => void
  damageEnemy: (amount: number) => void
  healPlayer: (amount: number) => void
  setTurnPhase: (phase: TurnPhase) => void
  setCurrentQuestionType: (type: QuestionType) => void
  nextStage: () => void
  resetGame: () => void
}

export const useGameStore = create<GameState>((set) => ({
  playerHp: 100,
  playerMaxHp: 100,
  enemyHp: 100,
  enemyMaxHp: 100,
  stage: 1,
  turnPhase: 'idle',
  currentQuestionType: null,

  damagePlayer: (amount) =>
    set((s) => ({ playerHp: Math.max(0, s.playerHp - amount) })),
  damageEnemy: (amount) =>
    set((s) => ({ enemyHp: Math.max(0, s.enemyHp - amount) })),
  healPlayer: (amount) =>
    set((s) => ({ playerHp: Math.min(s.playerMaxHp, s.playerHp + amount) })),
  setTurnPhase: (phase) => set({ turnPhase: phase }),
  setCurrentQuestionType: (type) => set({ currentQuestionType: type }),
  nextStage: () =>
    set((s) => ({
      stage: s.stage + 1,
      enemyHp: 150,
      enemyMaxHp: 150,
      turnPhase: 'idle',
      currentQuestionType: null,
    })),
  resetGame: () =>
    set({
      playerHp: 100,
      playerMaxHp: 100,
      enemyHp: 100,
      enemyMaxHp: 100,
      stage: 1,
      turnPhase: 'idle',
      currentQuestionType: null,
    }),
}))
```

### 구현 규칙

- `immer` 사용 금지, 순수 Zustand로 구현
- `damagePlayer`: HP 0 미만 방지 (`Math.max(0, ...)`)
- `healPlayer`: HP 최대 초과 방지 (`Math.min(playerMaxHp, ...)`)
- `gameover` / `stageclear` 전환은 이 스토어에서 하지 않음 → `useGameEngine` 담당

## 완료 기준

- [ ] `damagePlayer(200)` → `playerHp === 0` (음수 아님)
- [ ] `healPlayer(9999)` → `playerHp === playerMaxHp`
- [ ] `nextStage()` → `stage` 1 증가, `turnPhase === 'idle'`
- [ ] `resetGame()` → 초기값으로 완전 복원
- [ ] TypeScript 컴파일 에러 없음, `any` 없음

## Claude 코드 리뷰 체크리스트

- HP 0 미만 / 최대 초과 방어 로직 모든 액션에 있는가
- `GameState` 타입에서 액션이 전부 명시되어 있는가
- `any` 타입 없는가
- `nextStage`에서 enemyHp/enemyMaxHp가 스테이지 2 설정값으로 리셋되는가
