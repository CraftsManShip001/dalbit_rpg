## 목표

판정 결과와 HP 변화에 따른 시각적 피드백을 추가한다.
CSS `@keyframes` 우선, Framer Motion은 번들 사이즈 고려해서 사용 최소화.

## 구현할 이펙트

### 1. 데미지 숫자 팝업

HP 변화 시 숫자가 위로 떠오르며 사라지는 효과:

```css
@keyframes floatUp {
  0% { opacity: 1; transform: translateY(0); }
  100% { opacity: 0; transform: translateY(-40px); }
}
```

- 플레이어 HP 감소: 빨간 `-N`
- 적 HP 감소: 노란 `-N`
- 플레이어 HP 회복: 녹색 `+N`
- 지속 시간: 0.8s

구현 방법: 숫자 팝업을 state 배열로 관리 → 0.8s 후 배열에서 제거

```ts
type DamagePopup = { id: number; value: number; color: 'red' | 'yellow' | 'green'; x: number; y: number }
```

### 2. 판정 피드백 배너

정답/오답/타임아웃 결과 표시 (1.5초 후 자동 사라짐):

```css
@keyframes fadeInOut {
  0% { opacity: 0; transform: scale(0.8); }
  20% { opacity: 1; transform: scale(1); }
  80% { opacity: 1; }
  100% { opacity: 0; }
}
```

- 정답: 녹색 배경 "✓ 정답!"
- 오답: 빨간 배경 "✗ 오답"
- 타임아웃: 주황 배경 "⏱ 시간 초과"

### 3. 오답 화면 흔들림

```css
@keyframes shake {
  0%, 100% { transform: translateX(0); }
  20% { transform: translateX(-8px); }
  40% { transform: translateX(8px); }
  60% { transform: translateX(-4px); }
  80% { transform: translateX(4px); }
}
```

오답 시 BattlePage 최상위 div에 0.5s 적용.
