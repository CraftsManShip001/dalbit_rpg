# T03.5 — 디자인 시스템 (panda.config.ts)

## 전제 파일 (T03 완료)

- `panda.config.ts` 기본 설정 존재, 이 태스크에서 토큰 전부 채움
- `src/index.css` PandaCSS 레이어 import 존재

## 목표

RPG 던전 테마 디자인 시스템을 PandaCSS 토큰으로 정의한다.
이후 모든 컴포넌트는 이 토큰만 사용한다. 하드코딩 색상값 금지.

## 폰트 설정

Google Fonts 또는 CDN으로 아래 폰트 로드:
- **Pretendard** — 본문, UI 텍스트 전반
- **Paperlogy** — 타이틀, 게임 헤딩 (없으면 Pretendard로 대체)

`index.html`에 폰트 import 추가.

## Claude 코드 리뷰 체크리스트

- 하드코딩 색상값이 `panda.config.ts` 외부에 없는가
- `globalCss` body 배경/폰트 설정 있는가
- keyframes 이름이 T11 PRD 기준과 일치하는가 (`floatUp`, `fadeInOut`, `shake`)
- `any` 타입 없는가
