# antigravity 브라우저 테스트 프롬프트

## 사용법

태스크 구현 + Claude 리뷰 통과 후 antigravity에게 아래 형식으로 요청.

```
[브라우저 테스트 요청]

실행 명령: pnpm dev (http://localhost:5173)
태스크: T09 — YaksokEditor

테스트 항목:
1. 에디터에 한글 입력 시 조합 중 깨짐 없음 (가, 각, 강 순서로 입력)
2. 탭 키 입력 시 4칸 스페이스 삽입
3. disabled={true} 상태에서 입력 불가 확인
4. 1280px / 768px 에디터 레이아웃 깨짐 없음

보고 형식:
[항목] [결과: 통과/실패] - 실패 시 설명 + 스크린샷
```

---

## 공통 테스트 항목 (모든 브라우저 테스트에 포함)

```
[ ] 콘솔 에러 없음 (빨간 에러 없음, 경고는 기록)
[ ] 1280px 레이아웃 정상
[ ] 768px 레이아웃 정상
```

---

## 최종 E2E 테스트 (T12 완료 후)

```
시나리오: 스테이지 1 전체 플레이

1. 앱 시작 → 전투 화면 진입
2. 빠른 문제 출제 확인
3. 정답 코드 입력 → 실행 → 적 HP 감소 확인
4. 적 자동 공격 → 플레이어 HP 감소 확인
5. 오답 입력 → 효과 없음 확인
6. 알고리즘 문제 오답 → 플레이어 HP 감소 확인
7. 적 HP 0 → 스테이지 클리어 화면
8. [다음 스테이지] → 스테이지 2 진입
9. 플레이어 HP 0 → 게임오버 화면
10. [다시 시작] → 초기 상태 리셋

[ ] 모든 시나리오 통과
```
