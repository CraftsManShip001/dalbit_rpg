# 달빛약속 코딩 RPG — CLAUDE.md

## 프로젝트 개요

달빛약속(한국어 프로그래밍 언어)으로 코드를 작성해서 전투하는 교육용 RPG 웹앱.
플레이어가 에디터에서 달빛약속 코드를 작성하면 그 결과가 전투 행동이 된다.

## 기술 스택

| 도구 | 용도 | 비고 |
|------|------|------|
| React 18 | UI | |
| PandaCSS | 스타일 | CSS-in-JS 금지, PandaCSS만 |
| Vite | 번들러 | |
| @dalbit-yaksok/core | 달빛약속 런타임 | JSR 패키지. npm에 없음. 이미 설치 완료 (jsr:^6.0.5). import는 '@dalbit-yaksok/core'로 동일 |
| CodeMirror 6 | 코드 에디터 | |
| Zustand | 게임 상태 관리 | |

## 코딩 컨벤션

- TypeScript strict mode. `any` 타입 절대 금지
- 컴포넌트 파일: PascalCase (`BattlePage.tsx`)
- 훅 파일: camelCase (`useGameEngine.ts`)
- 인터페이스는 `I` 접두사 없이 (`GameState`, not `IGameState`)
- 인라인 style 속성 금지. PandaCSS css() 또는 styled() 사용
- 커밋 메시지: 한국어 (`feat: 게임 엔진 훅 구현`)
- 서비스 레이어와 UI 레이어 분리 유지

## 달빛약속 언어 필수 지식

### 기본 문법

```
# 출력
"안녕, 세상!" 보여주기

# 변수
나이 = 25
이름 = "달빛"

# 연산
10 + 3 보여주기    → 13
2 ** 3 보여주기    → 8
10 % 3 보여주기    → 1

# 조건문 (들여쓰기 반드시 4칸)
만약 점수 >= 60 이면
    "합격" 보여주기
아니면 만약 점수 >= 40 이면
    "재시험" 보여주기
아니면
    "불합격" 보여주기

# 반복
5번 반복
    "안녕" 보여주기

반복
    횟수 += 1
    만약 횟수 > 5 이면
        반복 그만

# 목록 — 인덱스 1부터 시작 (주의!)
음식 = ["피자", "햄버거", "파스타"]
음식[1] 보여주기    → 피자

# 약속 (함수)
약속, (반지름)으로 원의넓이 구하기
    반지름 * 반지름 * 3.14 반환하기
```

### 핵심 주의사항

- **들여쓰기는 반드시 4칸** (탭 아님)
- **목록 인덱스는 1부터 시작** (0 아님)
- 한글 키워드: `만약`, `이면`, `아니면`, `반복`, `반복 그만`, `반환하기`, `보여주기`

### 런타임 사용법

```ts
import { YaksokSession } from '@dalbit-yaksok/core'

const session = new YaksokSession()
session.addModule('main', `"안녕" 보여주기`)
await session.runModule('main')
```

**`보여주기` 출력 캡처 방법은 반드시 공식 문서 확인 후 구현할 것.**
추정으로 구현하지 말 것.

## 아키텍처 규칙

```
src/
├── pages/          # 라우트 단위 페이지
├── components/     # UI 컴포넌트 (game/, ui/ 하위 분류)
├── hooks/          # 게임 로직 커스텀 훅
├── workers/        # Web Worker (달빛약속 런타임 격리 실행)
├── data/           # 문제 데이터, 스테이지 설정 (순수 데이터)
└── store/          # Zustand 전역 상태
```

- 훅은 단일 책임. `useGameEngine`이 판정 로직까지 갖지 않는다
- 컴포넌트는 상태를 직접 갖지 않고 훅/스토어에서 가져온다
- `data/` 디렉터리는 순수 데이터만. 로직 없음

## 게임 규칙 (구현 기준)

| 유형 | 정답 판정 | 성공 효과 | 실패 효과 |
|------|-----------|-----------|-----------|
| 빠른 문제 | 출력값 일치 | 적 데미지 | 없음 |
| 따라치기 | 문자열 완전 일치 | HP 회복 or 방어막 | 없음 |
| 알고리즘 | 출력값 일치 | 필살기 데미지 | 내 HP 대량 감소 |

- 적은 N초마다 자동 공격 (문제 풀이 중에도 멈추지 않음)
- 무한루프 방지: 달빛약속 실행은 Web Worker + 5초 타임아웃 필수
- 타임아웃 발생 시 빠른문제/따라치기 → 효과 없음, 알고리즘 → HP 감소

## 에이전트에게 요청 시 금지사항

- 추정으로 달빛약속 API 사용하지 말 것 (문서 확인 후 구현)
- `any` 타입 사용 금지
- PandaCSS 외 스타일 방식 도입 금지
- Web Worker 없이 메인 스레드에서 달빛약속 실행 금지
